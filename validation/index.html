Validation and reproducibility

The project runs real RHF/UHF calculations, but it is an educational engine. This page separates tested numerical behavior from external experimental context and from qualitative toy models.

Current regression result

On 11 July 2026, npm test passed the complete scientific and SEO suite. The five-case browser benchmark passed with a common tolerance of 5×10⁻⁴ hartree:

  • H₂: calculated −1.116717 Ha; target −1.116700 Ha.
  • HeH⁺: calculated −2.841838 Ha; target −2.841800 Ha.
  • H₂O: calculated −74.965901 Ha; target −74.965900 Ha.
  • CH₄: calculated −39.726809 Ha; target −39.726900 Ha.
  • N₂: calculated −107.495893 Ha; target −107.496000 Ha.

These rounded targets are regression references at the listed geometries, method and basis. Agreement proves implementation consistency; it does not establish experimental accuracy.

What else is checked

  • RHF/UHF energies, electron counts, spin expectation values and the Coulombic virial ratio.
  • FCI correlation and the correct two-atom limit for stretched H₂ in the finite basis.
  • Dipole, Mulliken charge conservation and Mayer bond-order regression values.
  • 6-31G and 6-31G* basis behavior, including d functions and variational energy ordering.
  • Boys localization invariants, geometry optimization, numerical Hessians and IR behavior.
  • Provenance, report manifests, heuristic-risk labels, cancellation and cache-key normalization.

The executable assertions are public in test/selfcheck.js. Every push and pull request runs them in GitHub Actions.

Reproduce a result

  1. Open a URL that fixes mol, basis, mode and, if needed, lang.
  2. Record the geometry, charge, multiplicity, RHF/UHF method and basis from the exported REPORT or JSON manifest.
  3. Run npm test from a clean checkout to verify the numerical contracts.
  4. Use the Bridge template to repeat publication-grade work in PySCF or another established package.
Scope boundary.

Small-basis Hartree–Fock omits most electron correlation and is not a quantitative prediction method. The uncertainty panel is a qualitative heuristic, not a statistical confidence interval. Cavity, scaling and molecule-finder panels are explicitly toy or experimental models.

Citation

Cite the software using the repository’s machine-readable CITATION.cff. Record the commit hash and exported request key so another reader can identify the exact code and input.

Валидация и воспроизводимость

Проект выполняет настоящие расчёты RHF/UHF, но остаётся учебным движком. Здесь проверенное численное поведение отделено от внешних экспериментальных сведений и качественных toy-моделей.

Открыть проверенный пример H₂ Автоматические проверки

Текущий результат регрессии

11 июля 2026 года npm test прошёл полный набор научных и SEO-проверок. Браузерный benchmark из пяти молекул прошёл с общим допуском 5×10⁻⁴ хартри:

  • H₂: расчёт −1,116717 Ha; цель −1,116700 Ha.
  • HeH⁺: расчёт −2,841838 Ha; цель −2,841800 Ha.
  • H₂O: расчёт −74,965901 Ha; цель −74,965900 Ha.
  • CH₄: расчёт −39,726809 Ha; цель −39,726900 Ha.
  • N₂: расчёт −107,495893 Ha; цель −107,496000 Ha.

Это округлённые регрессионные цели для указанных геометрий, метода и базиса. Совпадение подтверждает согласованность реализации, но не экспериментальную точность.

Что ещё проверяется

  • Энергии RHF/UHF, число электронов, среднее ⟨S²⟩ и кулоновское вириальное отношение.
  • Корреляция FCI и правильный предел двух атомов для растянутой H₂ в конечном базисе.
  • Диполь, сохранение полного заряда по Малликену и регрессия порядков связи Майера.
  • Поведение базисов 6-31G и 6-31G*, включая d-функции и вариационный порядок энергий.
  • Инварианты локализации Бойза, оптимизация геометрии, численный гессиан и ИК-поведение.
  • Provenance, манифест отчёта, эвристические risk-метки, отмена задач и нормализация cache key.

Исполняемые проверки опубликованы в test/selfcheck.js. GitHub Actions запускает их для каждого push и pull request.

Как воспроизвести результат

  1. Откройте URL, фиксирующий mol, basis, mode и при необходимости lang.
  2. Сохраните геометрию, заряд, мультиплетность, RHF/UHF-метод и базис из REPORT или JSON-манифеста.
  3. Запустите npm test в чистой копии репозитория для проверки численных контрактов.
  4. Для публикационной работы повторите расчёт через шаблон Bridge в PySCF или другой признанной программе.
Граница применимости.

Хартри–Фок в малом базисе не учитывает основную часть электронной корреляции и не является количественным методом предсказания. Панель uncertainty — качественная эвристика, а не статистический доверительный интервал. Cavity, scaling и molecule finder явно обозначены как toy- или экспериментальные модели.

Цитирование

Цитируйте программу с помощью машиночитаемого файла CITATION.cff. Укажите hash коммита и request key из экспорта, чтобы читатель мог восстановить точный код и входные данные.
